# Fresh Schedules Marketing Website

This project is a complete, multi-page marketing website for "Fresh Schedules," a SaaS scheduling platform, built using the Next.js 14 App Router, React, TypeScript, and Tailwind CSS.

## Requirements

- `Node.js` 18+ (or 20+ recommended)
- `pnpm`

### Local Installation

```bash
pnpm install
pnpm run dev
```

Open `http://localhost:3000` in your browser.

### Deployment

Works on Vercel, Cloudflare Pages, Netlify.

Build command: `pnpm run build`
